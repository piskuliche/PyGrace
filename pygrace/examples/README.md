# Examples

Launch the sample 3-line plot:

```bash
./examples/open_example_plot.sh
```

Launch the error-bar demo plot:

```bash
./examples/open_errorbar_plot.sh
```

Sample datasets live in `examples/data/`:
- `rising.dat`
- `falling.dat`
- `bowl.dat`
- `errorbars_y.dat`
- `errorbars_xy.dat`
- `errorbars_y.csv`
